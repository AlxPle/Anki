# Anki — Deutsch-Russisch Karten / Карточки немецкий-русский

Этот репозиторий содержит текстовые материалы для создания карточек Anki по немецкому языку.
Dieses Repository enthält Textmaterialien zur Erstellung von Anki-Karteikarten für Deutsch.

## Что это / Was ist das?
- Списки слов и фраз на немецком языке для обучения.
- Wort- und Satzlisten auf Deutsch zum Lernen.
- Материалы для карточек с переводом (немецкий ↔ русский).
- Materialien für Karteikarten mit Übersetzung (Deutsch ↔ Russisch).

## Для чего / Wofür?
- Быстро собирать колоды Anki для регулярного повторения.
- Anki-Decks schnell für regelmäßige Wiederholung erstellen.
- Учить базовую лексику и грамматику по шагам.
- Grundwortschatz und Grammatik schrittweise lernen.

## Структура / Struktur
- `Schritte plus Neu 1.txt` … `Schritte plus Neu 4.txt` — части курса / Kursteile.
- `Schritte plus neu Glossar 1-2.txt`, `Schritte plus neu Glossar 3-4.txt` — глоссарии / Glossare.
- `Schritt für Schritt zum DTZ.txt` — подготовка к DTZ / DTZ-Vorbereitung.
- `Top 100 nouns.txt`, `Top 100 verbs.txt`, `Top 100 adjectives.txt` — частотные списки / häufige Wörter.

## Как использовать / Verwendung
1. Откройте нужный `.txt` файл / Öffne die gewünschte `.txt`-Datei.
2. Выберите формат карточки (слово, перевод, пример) / Wähle das Kartenformat (Wort, Übersetzung, Beispiel).
3. Импортируйте данные в Anki и настройте повторение / Importiere die Daten in Anki und passe die Wiederholungen an.

## Примечание / Hinweis
Файлы хранятся в простом текстовом формате для лёгкого редактирования.
Die Dateien sind im einfachen Textformat gespeichert und leicht bearbeitbar.
